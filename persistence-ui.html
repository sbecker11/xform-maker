About X-Form Storage

X-Forms are automatically saved to your browser's built-in database (IndexedDB).

  • Save Current: Save the current X-Form state to the database
  • Import: Load X-Forms from JSON files
  • Export All: Save all X-Forms to a single JSON file (backup)
  • Individual Export: Use the download button (⬇️) on any X-Form to save it as a file

Note: All X-Forms are stored locally in your browser. Clearing browser data will delete them.
